This project is an IoT-powered smart basket system that ensures farm-to-market transparency for organic fruits.
Each basket is sealed with an IoT lock and a unique QR code, allowing only authorized users (farmers, officials, or shopkeepers) to open it.
All sealing and unlocking events are recorded with date, time, and user details on a web dashboard, preventing tampering and fraud.
